Política de Privacidade
1. Introdução

Bem-vindo à Política de Privacidade do nosso bot no Discord. Nós nos preocupamos com a sua privacidade e estamos comprometidos em proteger as informações pessoais que você compartilha ao usar o nosso bot. Esta política explica como coletamos, usamos e protegemos os dados que você fornece.

2. Informações Coletadas

Ao utilizar o nosso bot no Discord, podemos coletar as seguintes informações:

Identificação do Usuário: O bot coleta o ID de usuário do Discord e o nome de usuário para poder identificar você e interagir de forma personalizada.

Mensagens Enviadas: O bot pode coletar as mensagens enviadas por você no chat do servidor ou em comandos direcionados ao bot. Essas informações são utilizadas para processar suas solicitações e fornecer respostas adequadas.

Comandos e Interações: Quando você utiliza comandos específicos do bot, como !ajuda ou !ranking, podemos coletar dados sobre esses comandos para entender como o bot está sendo utilizado e melhorar a experiência do usuário.

3. Como Usamos as Informações

As informações coletadas são usadas de diversas maneiras para garantir que o bot funcione corretamente e ofereça uma experiência útil:

Respostas e Funcionalidades: Usamos os dados para responder aos seus comandos e interagir com você de maneira personalizada.

Melhoria do Bot: A coleta de dados sobre interações e mensagens ajuda a entender como os usuários estão utilizando o bot, permitindo que possamos melhorar suas funcionalidades e corrigir problemas técnicos.

Segurança e Monitoramento: Em caso de problemas técnicos ou abusos do bot, podemos usar essas informações para garantir o bom funcionamento e a segurança do serviço.

4. Proteção dos Dados

A segurança das suas informações é importante para nós. Tomamos medidas para proteger os dados que coletamos contra acessos não autorizados, alterações ou destruição indevida. No entanto, é importante lembrar que, embora façamos o nosso melhor para proteger suas informações, nenhum sistema de segurança é 100% seguro, e não podemos garantir a total segurança de dados transmitidos pela internet.

5. Compartilhamento de Dados

Nós não compartilhamos as informações coletadas com terceiros, exceto nos seguintes casos:

Obrigação Legal: Podemos divulgar informações se for exigido por lei ou se acreditarmos que é necessário para cumprir com um processo legal, proteger nossos direitos ou garantir a segurança dos usuários.

Serviços de Terceiros: Se utilizarmos serviços de terceiros para realizar funções específicas no bot (por exemplo, análise de dados ou envio de mensagens), tomaremos medidas para garantir que esses serviços sigam as mesmas normas de privacidade.

6. Consentimento

Ao utilizar nosso bot, você concorda com a coleta e o uso das suas informações conforme descrito nesta Política de Privacidade. Se você não concordar com esta política, pedimos que não utilize o bot.

7. Alterações na Política de Privacidade

Esta Política de Privacidade pode ser atualizada periodicamente. Quando houver alterações significativas, avisaremos os usuários de forma clara, por meio de um aviso no servidor do bot ou na interface do bot.
